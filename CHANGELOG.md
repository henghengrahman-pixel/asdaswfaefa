# Changelog

## 1.1.0
- Nama produk selalu tampil pada setiap kartu katalog sebagai teks HTML.
- Harga, harga lama, rating, jumlah terjual, dan tombol detail selalu tampil.
- Admin dapat memasukkan beberapa URL foto sekaligus, satu URL per baris.
- Maksimal 12 foto per produk; foto pertama menjadi foto utama.
- Foto lama dapat dipilih dan dihapus saat mengedit produk.
- Upload beberapa file foto tetap tersedia.
- Validasi URL foto dan fallback gambar rusak ditambahkan.
- Pemeriksaan sintaks, kompilasi 25 template EJS, unit test, dan HTTP smoke test lulus.
